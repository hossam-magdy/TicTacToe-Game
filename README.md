TicTacToe
=========

A Symfony project created on September 4, 2017, 5:20 pm.

#### To run:
- Download/clone this repo & cd
- Run: `composer install`
- Run: `php bin/console server:run`
- Open: [http://localhost:8000/](http://localhost:8000/)
